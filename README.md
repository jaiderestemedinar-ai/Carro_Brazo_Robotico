# 🚗 Proyecto Semillero: Primera Parte-Jaider Medina

Este repositorio documenta el progreso, simulaciones y código para el desarrollo de un vehículo omnidireccional integrado con un brazo robótico.

## 🎯 Meta del Proyecto
Lograr la integración funcional entre un chasis de tracción omnidireccional y un brazo robótico, garantizando control inalámbrico independiente, simulaciones precisas y un diseño mecánico optimizado.

---

## 📋 Índice y Plan de Trabajo

### 1. Sistema de Tracción Omnidireccional (Chasis)
- [ ] Simulación de cinemática del vehículo en CoppeliaSim.
- [ ] Validación del movimiento lateral y diagonal ("tipo cangrejo") con llantas Mecanum.
- [ ] Programación y control de motores DC utilizando el microcontrolador PIC18F45k22.

### 2. Brazo Robótico (Basado en xArm 1s)
- [ ] Diseño en SolidWorks de un nuevo gripper (pinza) más liviano para optimizar la carga del brazo.
- [ ] Ensamblaje mecánico del brazo al chasis.
- [ ] Programación del control independiente utilizando el módulo ESP8266.

### 3. Sistema de Comunicación
- [ ] Configuración y establecimiento de red inalámbrica con módulos XBee.
- [ ] Pruebas de latencia y envío de tramas de datos entre los distintos controladores.

### 4. Documentación y Planos
- [ ] Subir esquemas electrónicos.
- [ ] Subir planos CAD de las piezas diseñadas.
